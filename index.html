Cours : Gérer son code avec Git et GitHub

Activité : Partagez un repository sur GitHub

Description du projet :

  1. Créer un repository contenant : un fichier README.txt qui présente le projet et au moins 2 autres fichiers de notre choix
  2. Faire au mois 4 commits distincts montrant l'ajout et la modifcations de certains fichiers, en soignant les messages de commits
  3. Envoyer le repository sur GitHub et copier l'URL du repository créé pour spécialement pour l'activité
  4. Télécharger le document .txt fourni dans l'énoncer, y coller l'URL du repository, l'enregistrer et le compresser dans un fichier .zip
  5. Envoyer le fichier .zip pour qu'il soit examiner par les autres étudiant.
